Create a regional subnet

It is required to have a regional subnet in the VCN where the API Gateways will be created. Click Create Subnet, enter a name, and select Regional (Recommended). Specify a CIDR block that provides a minimum of 32 free IP addresses, and select Public Subnet. Accept default values for other subnet properties, and click Create Subnet.:

Add a new Stateful Ingress rule

Add a new stateful ingress rule for the regional subnet to allow traffic on port 443. Click Security Lists, then click the name of the Default Security List for the VCN, then click Add Ingress Rules. Specify:

        Source Type: CIDR

        Source CIDR: 0.0.0.0/0

        IP Protocol: TCP

        Source Port Range: All

        Destination Port Range: 443

        Click Add Ingress Rules to add the new rule to the default security list.

@property

@property

return self._name_space

 

@property

# Load the default configuration

monitoring_client = monitoring.MonitoringClient(config)

def get_metrics(self):

compartment_id=self.compartment_id,

return response
